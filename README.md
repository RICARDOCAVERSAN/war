1. Usando as ferramentas:
- NodeJS v20 LTS
- AdonisJS 6
- MySQL
- Axios
- React
- Bootstrap v5.3


2. Projeto:

- Permite marcar e desmarcar um personagem como favorito
- Exibe os favoritos
